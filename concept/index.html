WHAT IS PROJECT FESTA?

未来を創る、
知の冒険。

Sharing Dreams - 夢を共有し、未来へ繋ぐ -

「問い」から始まる、
新しい世界への扉。

Project Festa 2026は、単なる文化祭ではありません。それは、生徒一人ひとりが抱く「なぜ？」という問いを出発点とし、一年間かけて深めた探究の成果を発表するアカデミックな祭典です。

私たちの世界は、答えのない問いで溢れています。教科書の中にある正解を覚えるだけでなく、自ら課題を見つけ、仮説を立て、検証し、そして新たな価値を創造する。そのプロセスこそが、未来を生き抜く力となります。

「Sharing Dreams」というテーマには、個々の夢や発見を共有し合うことで、新たな化学反応を起こしたいという願いが込められています。教室から社会へ、そして未来へ。生徒たちの情熱が交錯する瞬間を、ぜひ体感してください。

MESSAGE

教員からのご挨拶

Project Festa 2026へようこそ。

本校では「自ら考え、自ら行動し、自ら創り出す」力を育むことを教育の柱としています。このフェスタは、生徒たちが日々の学習活動の中で培ってきた思考力や表現力を、社会に向けて発信する最大の機会です。

失敗を恐れず挑戦した記録、仲間と議論を重ねて導き出した結論、そして未完成ながらも輝きを放つアイデアの数々。ここにあるのは、完成された展示物だけではなく、彼らの成長の軌跡そのものです。

生徒たちの熱意ある発表に耳を傾け、ぜひ対話を楽しんでください。皆様との交流が、彼らの学びをより深いものへと進化させます。
